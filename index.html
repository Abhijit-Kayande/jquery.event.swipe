Swipe events

swipeleft, swiperight, swipeup, swipedown
Fired when the finger is released from the touch surface if a) the finger has travelled a distance or b) the finger has been swiped quickly.

Swipe event objects are augmented with the properties:

e.pageX, e.pageY
Current page coordinates of pointer.
e.startX, e.startY
Page coordinates the pointer had at the tart of the swipe.
e.distX, e.distY
Distance finger moved during the swipe.
e.velocityX, e.velocityY
Velocity, in pixels/ms, averaged over the last few move events.

How to use swipe events

{% highlight js %} jQuery('.mydiv') .on('swipeleft', function(e) { // View the next slide jQuery('.next_slide').addClass('active'); }) .on('swiperight', function(e) { // View the previous slide jQuery('.prev_slide').addClass('active'); });{% endhighlight %}

What if I want to scroll, not swipe?

Avoid messing with gesture events by preventing movestart:

{% highlight js %} jQuery('.mydiv') .on('movestart', function(e) { // move .mydiv horizontally jQuery(this).css({ left: e.startX + e.deltaX }); }) .bind('swipeup', function() { // move is complete! });{% endhighlight %}
